# Diseño de experiencia de usuario e interfaz de aplicación de seguridad para la mujer en Guatemala

## Descripción
[Breve descripción del proyecto: qué problema resuelve y cómo funciona]
Este proyecto consiste en una aplicación móvil orientada a la prevención y combate de la violencia, brindando a las personas usuarias una herramienta segura, accesible y fácil de usar para informarse, identificar situaciones de riesgo y buscar apoyo.
La aplicación funciona centralizando recursos clave, ofreciendo orientación clara y permitiendo el acceso rápido a información y acciones de apoyo, con el objetivo de empoderar a las personas, reducir barreras de acceso a ayuda y utilizar la tecnología como un medio de innovación social.

## Tecnologías Utilizadas
- Figma: diseño UX/UI, wireframes, prototipos interactivos y validación visual de la experiencia de usuario.
